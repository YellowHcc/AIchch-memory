# 🤖 AIchch 记忆备份系统

这个仓库用于备份 AIchch 的记忆文件和项目文件。

## 📁 目录结构

```
.
├── memory/              # 每日记忆日志
│   └── YYYY-MM-DD.md
├── MEMORY.md            # 长期记忆摘要
├── projects/            # 项目文件
│   └── notes/           # 便签网页
│       └── index.html
├── README.md            # 本文件
└── .github/
    └── workflows/       # 自动备份工作流
        └── backup.yml
```

## 🔄 自动备份

记忆文件会自动同步到这个仓库。

- **便签网页** - 一个本地存储的便签应用
